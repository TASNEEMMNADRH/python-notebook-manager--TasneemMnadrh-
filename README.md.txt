#Personal Notebook Manager

This is a Python console-based notebook application.
You can add, view, search, filter, edit, delete and sort notes.

 How to run#
Use the menu and choosing a number from the options:
(1) Add a new note
(2) List all notes
(3) Search notes
(4) Filter by tag
(5) Edit a note
(6) Delete a note
(7) Sort notes
(0) Exit
